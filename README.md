# SmartBuy Supermarket Billing System

## Project Description

The SmartBuy Supermarket Billing System is a Python-based application designed to automate customer billing processes. The system allows cashiers to enter product details, calculate bills, apply discounts automatically, and generate customer receipts efficiently.

## Features

- Multiple product entry
- Automatic bill calculation
- 10% discount for bills above Le 500
- Receipt generation
- Multiple customer processing
- User-friendly interface

## How the Program Works

1. The cashier enters the number of products purchased.
2. Product details (name, quantity, and price) are entered.
3. The system calculates the total cost of each product.
4. The overall bill is calculated automatically.
5. A 10% discount is applied if the total bill exceeds Le 500.
6. The final amount payable is displayed.
7. A receipt is generated for the customer.
8. The system can process multiple customers.

## Requirements

- Python 3.x

## How to Run the Program

1. Save the Python code as `billing_system.py`.
2. Open a terminal or command prompt.
3. Navigate to the project folder.
4. Run the program using:

```bash
python billing_system.py
```

## Problem Solved

This system eliminates manual billing errors, reduces waiting time at checkout counters, improves accuracy, and enhances customer satisfaction.
